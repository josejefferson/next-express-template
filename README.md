# Next.js + Express Template
Template full-stack utilizando **Next.js** como front-end, **Express.JS** como back-end e **MongoDB** como banco de dados

## Funcionalidades

- Sistema de login
- Gerenciamento de usuários
- Sistema de permissões
- Troca de senha
- Light/dark mode
- Atalhos de teclado
